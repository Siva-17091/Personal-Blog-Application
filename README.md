## Personal Blog Application Using Java and SpringBoot

I've build Personal Blog Application using Java and SpringBoot. You use this application without User Account for View Purpose and addition you can also create
user account and view. By, Accessing Admin control you can able to Manage the Post By Creating, Updating and Deleting Blog Post.

## Technologies Used:

Front-End: HTML, CSS.

Frameworks: BootStrap 5 and SpringBoot 3.

Programming Language: Java 17.

Spring Dependencies: Spring Web, Data JPA, Dev Tools, Validation, MY SQL Driver
Thymeleaf, Common-IO, Lombok (Getter and Setter).

Database: MY SQL.

Containerization Tools: Docker.
